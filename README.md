## About The Project

In this web app, you can add your things to do by writing them one by one in the input field and clicking the add button. 
When the task is completed, you can click on it in the list and it will be marked as completed. 
If you need to clear the entire list, just click on the delete button.

This is a test web app, created in the process of study React.
